# Reference Documentation

This directory holds authored reports and generated evidence that are useful for implementation work but too large or detailed for active AI onboarding.

## Areas

- `packets/`: packet structure reports, stripdown comparisons, and protocol gap notes.
- `gameplay/`: broad gameplay condition reports.
- `enemies/`: enemy, NPC, client RDB, and spawn/corpse visual references.
- `enemies/data/`: generated enemy evidence CSV files.
- `loot/`: mob loot notes and coverage data.
- `client-dll-function-map/`: generated AO client DLL symbol/function maps.

Keep source authority labels intact when using these files.
