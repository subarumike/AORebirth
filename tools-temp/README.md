# tools-temp

This directory is local development tooling only and is not part of the AO Rebirth server distribution.

AOSharp, EasyHook, capture injectors, generated captures, staging SQL, and smoke harness scripts in this tree are used as local reference or validation tools. They are intentionally outside `AORebirth/AORebirth.sln` and must not be packaged as AO Rebirth runtime code without a separate license review.
